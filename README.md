# tw-stock

# How to install:

```js
npm install -g tw-stock
```

# How to use:

```sheel
tw-stock stock <stock_code>
```

`<stock_code>` input your search tw_stock_code

| Options     | Description                            |
| ----------- | -------------------------------------- |
| -l --listed | input this tw_stock_code is tse or otc |
